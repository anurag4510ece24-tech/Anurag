# RTOS Experiments

This repository contains Real-Time Operating System (RTOS) experiments and embedded system programs developed using FreeRTOS concepts. The experiments demonstrate task scheduling, multitasking, synchronization, inter-task communication, and peripheral interfacing on embedded hardware platforms.

---

## Repository Structure

| Folder Name | Description |
|-------------|-------------|

| Exp2_LED_Blinking_PushButton | LED control using push button with RTOS |

## Objectives

- Understand RTOS fundamentals
- Learn task creation and scheduling
- Implement multitasking in embedded systems
- Study synchronization techniques
- Interface hardware peripherals using RTOS
- Analyze real-time behavior of tasks

---

## Technologies Used

- Embedded C
- FreeRTOS
- STM32 Microcontroller
- STM32CubeIDE
- HAL Drivers

---

## Features

- Task Scheduling
- LED Blinking using Tasks
- Push Button Interfacing
- Delay and Timing Functions
- Inter-task Communication
- Queue and Semaphore Handling
- Real-Time Task Management

---

## Software Requirements

- STM32CubeIDE
- Keil uVision (optional)
- FreeRTOS Library

## Hardware Requirements
